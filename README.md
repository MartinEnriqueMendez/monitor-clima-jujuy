# Monitor Climático para Agro 4.0 - Jujuy

Sistema de extracción de datos climáticos en tiempo real para la zona de San Pedro de Jujuy/La Mendieta. 

## 🛠️ Tecnologías
- **Lenguaje:** Python 3.x
- **Base de Datos:** SQLite3
- **API:** OpenWeatherMap
- **Entorno:** Virtualenv + Dotenv (Seguridad de API Keys)

## 🚀 Funcionalidad
El script consulta la temperatura y humedad actual, procesa la información y la almacena en una base de datos local para su posterior análisis de productividad agrícola.
